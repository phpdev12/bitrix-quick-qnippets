# bitrix-quick-qnippets

Коллекция коротких полезных скриптов для быстрого копирования и вставки. Решаю повседневные задачи — от обработки текста до автоматизации рутинных операций.

## create_notification_event_and_template.php

**Назначение:**  
Создание типа почтового события и шаблона для уведомления клиента об успешной отправке формы (Bitrix).

**Функциональность:**  
- регистрирует событие `CLIENT_FORM_SUCCESS_NOTIFICATION`;  
- связывает событие с текстовым шаблоном письма.
